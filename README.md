#####7/27总结
>从头开始再次回头看这个幻灯片制作，本质上就是将图片出现以及隐藏的两种状态用css分别表现出来
.show{
    display:show
}
.notshow{
    display:none
}
然后利用js获取图片所在的dom结构
<div class="slide show">
    <img src="images/1.jpg"/>
</div>
通过添加删除属性来获得图片显示隐藏的效果，同时还将图片替换实现幻灯片效果
通过属性的切换来实现状态的改变。
关于框架部分，更牛逼搭好了结构只需要将不同的数据通过遍历代入即可
